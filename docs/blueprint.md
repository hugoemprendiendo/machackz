# **App Name**: HelloDisplay

## Core Features:

- Simple Greeting: Displays the text 'Hola' on the screen.

## Style Guidelines:

- Background color: Soft beige (#F5F5DC) for a warm, inviting feel.
- Primary color: Gentle brown (#A67B5B), providing a natural and calming impression.
- Accent color: Pale salmon (#F08080), a touch warmer and slightly more saturated than the other colors to draw some attention.
- Font: 'PT Sans' for a modern and readable text display (sans-serif).
- Center the greeting text on the page for simple readability.
